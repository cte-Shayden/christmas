Steam Deck

A portable PC gaming handheld. Great for playing PC titles on the go. Useful only if a desktop isn't available, but excellent as a secondary portable device.

Why I want it

Provides the flexibility to play PC games while away from a desk. Useful for testing game builds, portability, or playing less-demanding titles without taxing the desktop.

Key Features

  • Handheld form factor with ergonomic controls
  • Runs many PC games via SteamOS/Proton
  • Dockable to a monitor/TV

Considerations

  • Battery life varies by game
  • Not all games are fully compatible without tweaks
  • Storage options vary by model

Priority

Medium — great to have, but lower priority than the main gaming PC.
